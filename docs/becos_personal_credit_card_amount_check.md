# BECOS 개인신용카드사용액 확인 결과

확인일: 2026-06-23

대상 URL:

```text
http://www.becos.kr/db/stats01.asp?cate=stats&idx=114&stats=09
```

## 1. 결론

BECOS `개인신용카드사용액`은 금액 지표가 맞다. 단위도 `백만원`으로 설정되어 있다.

다만 BECOS HTML에 노출된 ECOS 호출 경로는 현재 ECOS 공식 API에서 그대로 조회되지 않는다.

```text
http://ecos.bok.or.kr/api/StatisticSearch/{API_KEY}/xml/kr/1/30/043Y070/MM/StartDay/EndDay/B/1000/TOT/
```

확인된 메타:

| 항목 | 값 |
|---|---|
| BECOS idx | `114` |
| 지표명 | 개인신용카드사용액 |
| 단위 | 백만원 |
| 원천 | 한국은행 |
| BECOS 노출 ECOS 통계코드 | `043Y070` |
| ECOS 항목 경로 | `B/1000/TOT` |

ECOS 인증키로 공식 API를 직접 확인한 결과, 현재 조회 가능한 지역별 개인 신용카드 통계는 `601Y002`다.

| 항목 | 값 |
|---|---|
| ECOS 통계코드 | `601Y002` |
| 통계명 | 지역별 소비유형별 개인 신용카드 |
| 부산 코드 | `B` |
| 소비유형 합계 코드 | `1000` |
| 금액구분 총액 코드 | `TOT` |
| 단위 | 백만원 |
| 주기 | 월 |
| 수록 시작 | 2009-12 |
| 수록 종료 | 2023-08 |

## 2. 공식 API 확인값

부산 합계 총액 조회 경로:

```text
601Y002/M/202301/202308/B/1000/TOT
```

| 기준월 | 부산 개인 신용카드 총액 |
|---|---:|
| 2023-01 | 2,085,010 백만원 |
| 2023-02 | 1,944,822 백만원 |
| 2023-03 | 2,151,290 백만원 |
| 2023-04 | 2,077,433 백만원 |
| 2023-05 | 2,148,488 백만원 |
| 2023-06 | 2,096,725 백만원 |
| 2023-07 | 2,155,108 백만원 |
| 2023-08 | 2,178,269 백만원 |

2023년 8월 부산 총액은 `2,178,269백만원`, 즉 약 `2조 1,783억원`이다.

월간 일평균 조회 경로:

```text
601Y002/M/202301/202308/B/1000/DAV
```

2023년 8월 부산 월간 일평균은 `70,267백만원`, 즉 약 `703억원/일`이다.

## 3. 운영 판단

| 항목 | 판단 |
|---|---|
| 금액 여부 | 가능. 단위 `백만원` |
| 월별 여부 | 가능 |
| 자동 수집 | 가능. 한국은행 ECOS OpenAPI |
| 부산 지역성 | 확인됨. `ITEM_CODE1=B`, `ITEM_NAME1=부산` |
| 최신성 | 부족. `END_TIME=202308` |
| 대시보드 채택 | 운영 본편 제외, 과거 참고지표 |

## 4. 권장 처리

이 지표는 아래 중 하나로 처리한다.

1. 최신 운영지표는 Nowcast `가맹점 카드매출액 변동률`, `신용카드 이용금액 변동률`을 사용한다.
2. 과거 금액 규모 설명이 필요하면 ECOS `601Y002`를 참고자료로 사용한다.
3. 화면에는 `수록 종료: 2023-08`을 명확히 표시한다.

화면 표기 금지:

```text
부산 개인신용카드 사용액 최신값
```

권장 표기:

```text
부산 지역별 소비유형별 개인 신용카드 금액
출처: 한국은행 ECOS
통계코드: 601Y002
수록 종료: 2023-08
단위: 백만원
```
