# 부산 Big-데이터웨이브 상황판 카드소비액 확인

확인일: 2026-06-23

## 결론

부산시 Big-데이터웨이브 상황판에는 월별 부산 카드소비액 금액 데이터가 있다.

- 화면 URL: `https://data.busan.go.kr/bdip/literacy/situation/dashboard.do`
- 실제 iframe: `https://bs.webasp.kr/dashboard`
- 데이터 API: `GET https://bs.webasp.kr/api/dashboard/eap?region=26&date=YYYYMM`
- 기간 API: `GET https://bs.webasp.kr/api/dateRangeInfo?region=26`
- 부산 시도 코드: `26`
- 확인된 수록범위: `2023-06~2026-04`
- 단위: `원`
- 자료출처: 상황판 매뉴얼 기준 `BC카드`

## 예시 응답

요청:

```text
GET https://bs.webasp.kr/api/dashboard/eap?region=26&date=202604
```

주요 값:

| 필드 | 값 | 해석 |
|---|---:|---|
| `card_spend.current` | 82,933,157,226 | 2026년 4월 부산 카드소비액 |
| `card_spend.prevMonth` | 79,886,705,990 | 2026년 3월 부산 카드소비액 |
| `card_spend.prevYear` | 73,956,201,727 | 2025년 4월 부산 카드소비액 |
| 전월 대비 | +3.81% | 파생 계산 |
| 전년 대비 | +12.14% | 파생 계산 |

`daily_card_spend.202604`의 시간대별 합계도 `82,933,157,226`으로 `card_spend.current`와 일치한다.

## 매뉴얼 근거

`manual_busan.pdf` 기준:

- Big-데이터웨이브 상황판은 `KT 통신 데이터`와 `카드사의 신용카드 매출 데이터` 기반 서비스다.
- 소비패턴 신용카드 데이터의 자료출처는 `BC카드`로 표기되어 있다.
- 소비 금액 단위는 `원`이다.
- 데이터 레벨은 `시도`, `시군구`, `행정동`이다.

## 운영 판단

채택 가능성은 높다. 우리가 찾은 공개 후보 중 부산 카드소비액을 원 단위로 월별 조회할 수 있는 가장 직접적인 후보다.

제약:

- 공식 공공 OpenAPI 문서가 아니라 화면용 API다.
- 호출 제한, 재사용 권한, 상업/공공서비스 재표출 가능 여부는 부산시 또는 상황판 운영기관 확인이 필요하다.
- 원자료가 BC카드 기반이므로 전체 카드시장 또는 부산 전체 민간소비를 완전 대표한다고 표현하면 안 된다.
- 대시보드 문구는 `BC카드 기반 부산 카드소비액` 또는 `부산 카드소비액(BC카드 기반)`으로 표기하는 편이 안전하다.

권장 표출:

- 상단 보조 카드: `부산 카드소비액`
- 값: 최신월 금액을 `억원` 단위로 환산
- 보조 지표: 전월 대비, 전년 동월 대비
- 상세 패널: 성·연령별 소비, 업종별 소비 TOP5, 시간대별 월별 추이
